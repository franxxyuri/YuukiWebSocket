# 🎯 从这里开始！

## 第一次使用？按照这个顺序操作：

### 1️⃣ 安装依赖（只需要做一次）

打开命令行（在项目根目录），运行：

```bash
npm install
```

等待安装完成（可能需要几分钟）。

---

### 2️⃣ 启动服务

**方式 A: 使用统一 CLI（推荐）**

```bash
wac-cli
```

会看到一个菜单，选择 `2` (开发模式)

**方式 B: 直接命令**

```bash
wac-cli dev
```

**方式 C: 使用 npm**

```bash
npm start
```

---

### 3️⃣ 验证启动成功

你应该看到类似这样的输出：

```
═══════════════════════════════════════════════════════════
🚀 高可用服务器启动成功！
═══════════════════════════════════════════════════════════
📡 后端服务: http://192.168.x.x:8928
⚡ 前端服务: http://192.168.x.x:8781
🔍 健康检查: http://192.168.x.x:8928/health
📊 性能监控: http://192.168.x.x:8928/api/performance
═══════════════════════════════════════════════════════════
```

---

### 4️⃣ 打开浏览器

访问以下任一地址：

- http://localhost:8781
- http://127.0.0.1:8781
- http://你的IP地址:8781

---

### 5️⃣ 连接 Android 设备

1. 在 Android 设备上安装并打开应用
2. 确保 Android 设备和电脑在同一个局域网
3. 应用会自动发现电脑
4. 点击连接

---

## 🎮 常用命令

```bash
wac-cli dev        # 启动开发服务器
wac-cli start      # 启动生产服务器
wac-cli stop       # 停止服务
wac-cli restart    # 重启服务
wac-cli status     # 查看状态
wac-cli test       # 运行测试
wac-cli help       # 查看帮助
```

---

## ❓ 遇到问题？

### 问题 1: 端口被占用

```bash
wac-cli stop
```

然后重新启动。

### 问题 2: 依赖安装失败

```bash
npm cache clean --force
npm install
```

### 问题 3: 找不到 wac-cli 命令

直接使用：

```bash
npm start
```

### 问题 4: 配置错误

运行测试检查配置：

```bash
npm test
```

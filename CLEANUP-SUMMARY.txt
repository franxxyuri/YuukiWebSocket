================================================================================
                            快速开始
================================================================================

1. 安装依赖:
   npm install

2. 启动开发环境:
   start-dev.bat
   
   或手动启动:
   npm run dev

3. 访问应用:
   前端: http://localhost:5173
   后端: http://localhost:3000

4. 运行测试:
   run-tests.bat
   
   或手动运行:
   npm test

5. 停止服务:
   stop-server.bat

================================================================================
                            主要功能
================================================================================

✅ 文件传输
   • 支持大文件（最大 10GB）
   • 断点续传
   • SHA-256 完整性验证
   • 实时进度更新

✅ 屏幕镜像
   • 30 FPS 实时流传输
   • H.264 压缩
   • 自适应质量调整
   • 自动重连机制

✅ 远程控制
   • 鼠标和键盘输入
   • 触摸手势识别
   • 事件批处理
   • 低延迟传输

✅ 性能优化
   • 连接池管理
   • 内存优化
   • CPU 使用率控制
   • 带宽自适应

                            常用命令
================================================================================

开发:
  npm start              - 启动后端服务器
  npm run dev            - 启动后端开发模式
  npm run dev:vite       - 启动前端开发服务器

  npm run android-build  - 构建 Android 应用
